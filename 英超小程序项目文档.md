# 英超资讯小程序 - 项目文档

> 一个专注于英超联赛的微信小程序，提供积分榜、赛程、球队和球员信息。

---

## 📋 项目概述

| 项目 | 说明 |
|------|------|
| 项目名称 | Premier League MiniApp |
| 技术栈 | 微信小程序原生 |
| 数据源 | football-data.org (免费版) |
| 目标平台 | 微信小程序 |

---

## 🎯 核心功能

### MVP 版本功能清单

- [x] **首页** - 焦点比赛、最新资讯、快捷入口
- [x] **积分榜** - 实时英超排名、球队战绩
- [x] **赛程** - 比赛日程、比分直播
- [x] **球队详情** - 球队资料、近期战绩、阵容
- [x] **球员详情** - 球员资料、赛季数据
- [x] **我的** - 关注球队、个人设置

## 🏗️ 技术架构

### 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|

## 🔌 数据层设计

### API 供应商

**football-data.org** - 免费版限制：10 次/分钟

- 官网：https://www.football-data.org/
- 注册获取 API Key

### API 接口清单

| 功能 | Endpoint | 缓存策略 |
|------|----------|---------|
| 积分榜 | `GET /v4/competitions/PL/standings` | 5分钟 |
| 赛程 | `GET /v4/competitions/PL/matches` | 1分钟 |
| 球队列表 | `GET /v4/competitions/PL/teams` | 1小时 |
| 球队详情 | `GET /v4/teams/{id}` | 10分钟 |
| 球员详情 | `GET /v4/persons/{id}` | 10分钟 |

### API 响应示例

#### 积分榜响应
```json
{
  "competition": { "name": "Premier League" },
  "standings": [{
    "stage": "REGULAR_SEASON",
    "table": [
      {
        "position": 1,
        "team": { "name": "Liverpool FC", "crest": "..." },
        "playedGames": 28,
        "won": 20,
        "draw": 7,
        "lost": 1,
        "points": 67
      }
    ]
  }]
}
```

## 📱 页面详细设计

### 1. 首页 (`pages/index/`)

**布局结构：**
```
┌─────────────────────────────┐
│  搜索栏          日期选择   │  ← 顶部操作区
├─────────────────────────────┤
│                             │
│      [ 焦点比赛轮播 ]       │  ← 轮播图
│                             │
├─────────────────────────────┤
│  [积分榜] [赛程] [球队]     │  ← 快捷入口
├─────────────────────────────┤
│  今日比赛                   │  ← 最新比分
│  ├── 曼联 vs 利物浦  2:1    │
│  └── 阿森纳 vs 曼城  1:1    │
└─────────────────────────────┘
```

**功能点：**
- 搜索栏支持搜索球队/球员
- 日期选择器快速切换比赛日
- 轮播展示焦点比赛（最近3场重要对决）
- 快捷入口跳转各模块
- 实时比分展示（已结束/进行中）

---

### 2. 积分榜 (`pages/standings/`)

**布局结构：**
```
┌─────────────────────────────┐
│  总榜 | 主场 | 客场          │  ← Tab切换
├─────────────────────────────┤
│  排名  球队     赛  胜  平  │
│  ─────────────────────────  │
│   1    利物浦   28  20  7   │
│   2    阿森纳   28  16  9   │
│   3    森林     28  15  6   │
│   ...                       │
└─────────────────────────────┘
```

**字段说明：**
- 赛：已赛场次
- 胜/平/负：胜负平场次
- 进/失：进球/失球
- 净：净胜球
- 分：积分

**交互：**
- 点击球队跳转详情页
- 下拉刷新数据

---

## ⚠️ 注意事项

### API 限制

1. **频率限制**：免费版 10 次/分钟，务必做好缓存
2. **数据时效**：比赛数据可能有 15-30 分钟延迟
3. **赛季限制**：休赛期 API 可能返回空数据

### 微信小程序规范

1. **域名配置**：所有接口域名需在小程序后台配置
2. **HTTPS**：必须使用 HTTPS 协议
3. **用户隐私**：获取用户信息需明确授权

### 图片资源

1. **队徽图片**：建议使用 CDN 或下载到本地
2. **懒加载**：列表图片使用 `lazy-load`
3. **占位图**：图片加载失败显示默认占位图
