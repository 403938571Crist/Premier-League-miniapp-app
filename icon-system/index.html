Icon System — 英超小程序

Rewritten from scratch. Tighter geometry on a 24u grid with 2u safe area, sub-pixel-aligned junctions, 1.6u stroke (finer than v1's chunky 1.75), proper football-specific detail (jersey V-neck + sleeve seams, trophy with real handles, whistle with lanyard hole). Every icon uses currentColor.

00v1 → v2 Refinement

A representative comparison. Same idea, refined execution.

v1

Single outline reads like a T-shirt. No collar. No number panel. Sleeves look glued on.

v2

V-neck defined, sleeves taper inward, small hem detail, and a two-line number placeholder. Reads as a football kit.
